P0 operational hints

  • VISUAL tags map detected visual channel context and need both channel bindings and rules.
  • MOOD/THEME tags are rules-only classification and do not use channel bindings.
  • Changing tags, rules, or bindings does not retroactively update already analyzed tracks; rerun Analyze or wait for future recompute.
  • Preview validates JSON and reports actions/errors without writing.
  • Confirm applies validated changes to the catalog.
