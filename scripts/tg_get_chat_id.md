# How to get your TG_ADMIN_CHAT_ID (simple)

1) Write to your bot in Telegram: /start
2) Temporarily run the bot with logging enabled (or add a small print in handlers)
3) In Telegram clients, easiest approach:
   - Create a private group, add the bot, send a message.
   - Use @RawDataBot or similar to get chat_id (manual).
